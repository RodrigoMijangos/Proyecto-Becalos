2. ¿Qué etiquetas semánticas usaste y por qué?
   - <header> para el encabezado principal del sitio. Contiene: <h1> (título del blog), <h2> (subtítulo) y <nav> (menú de navegación). Usadas para agrupar contenido introductorio.
   - <nav> para definir una sección de navegación. Contiene: <ul>, <li> y <a>. Sirve para indicar que esos enlaces son parte del menú principal del sitio.
   - <main> para representar el contenido principal del documento.
   - <section> para definir secciones temáticas del contenido. Cada sección tiene su propio <h2>.
   - <article> para definir contenido independiente y reutilizable. Aquí cada entrada del blog es un <article>, de modo que cada entrada podría mostrarse sola, tiene su propio <h3> y tiene su propio contenido.
   - <footer> para definir un <p> con copyright, que es información final del sitio.
